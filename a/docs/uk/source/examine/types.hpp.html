Розбір: a/types.hpp

Визначає fixed-width псевдоніми скалярних типів і базові інтегральні типи для всіх модулів a::.

Що Робить Цей Файл

  • Вводить канонічні цілочисельні псевдоніми на кшталт u8/u32/u64 та signed-варіанти.
  • Описує size-типи на кшталт usize/isize і константи типу npos.
  • Слугує базовим словником типів для всього шару.
