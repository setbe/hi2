Header/Source Split

  • Public API declarations stay in b/*.hpp and b/net/*.hpp.
  • Port contract declarations are in b/detail/port_contract.hpp.
  • Public wrappers are in b/*.cpp.
  • OS-specific implementations are in b/ports/<platform>/*.cpp.
